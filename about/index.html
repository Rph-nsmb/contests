About

NSMB(C) level contests is a Discord server created by Mariorrt and Rph. It aims to be a place where modders can submit contests and show their creativity by participating to them.

Note to NSMB Central's staff

This server has been created to allow unofficial contests, and it is its only purpose. We do not want to replace Central. We do not want to replace the staff members.
This server has been created because making constests hasn't been allowed. If you don't approve that, you can just not join the server. This server will not be used to criticize Central or its members.
This server aims to be a sub-server of Central. Therefore, the staff of NSMB Central has a control over the website and the server. They can take it down, but we adwise the staff to not do it.
